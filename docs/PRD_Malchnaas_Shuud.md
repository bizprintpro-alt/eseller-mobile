1. Товч танилцуулга (Executive Summary)
“Малчнаас шууд” нь eseller.mn платформ дээр хэрэгжүүлэх direct-from-herder маркетплейс юм. Энэ нь хөдөөгийн малчдыг хотын хэрэглэгчтэй шууд холбож, зуучлагчгүй зохиомжоор байгалийн цэвэр бүтээгдэхүүнийг (мах, ноос, арьс, сүү) худалдаалах боломж олгоно.
Зорилго: Монголын уламжлалт мал аж ахуйн бүтээгдэхүүнийг дижитал сувгаар зуучлагчгүй, шударга үнээр худалдах экосистем бий болгох.
2. Шийдэх асуудал (Problem Statement)
Монголын мах, ноос, арьс, сүү зэрэг бүтээгдэхүүн уламжлалт зах зээл дээр 4-5 шатлалтай нийлүүлэлтийн сүлжээгээр дамждаг:
Малчин → бэлтгэн нийлүүлэгч → бөөний худалдаачин → нэгтгэгч → жижиглэнгийн дэлгүүр → хэрэглэгч
Энэ явцад дараах асуудлууд гарна:
Малчин эцсийн үнийн 25-30%-ийг л авна. Ихэнх ашиг зуучлагчдад очно.
Хэрэглэгч 2-3 дахин үнэтэй бараа авч, гарал үүсэл нь тодорхойгүй.
Олон гараар дамжихад чанар буурах, хадгалалт муудах.
Органик, “малчнаас шууд” гэх таг нэрийг ашигласан хуурамч бараа зах зээлд олон байна.
Малчдад дижитал зах зээл нээлтгүй, тэд өөрсдөө шууд борлуулах суваггүй.
3. Зорилго ба амжилтын хэмжүүр
3.1. Бизнесийн зорилго
Малчдын орлогыг 40-60%-иар нэмэгдүүлэх (зуучлагч татгалзсаны үр дүнд).
Эцсийн хэрэглэгчийн үнийг 15-25%-иар бууруулах.
eseller.mn-ий ялгарах брэнд давуу тал (unique value proposition) бий болгох.
Монголын хөдөө орон нутгийн дижитал эдийн засгийн хөгжилд хувь нэмрээ оруулах.

6. Функционал шаардлагууд (Functional Requirements)
6.1. Малчны бүртгэл ба баталгаажуулалт
Утасны дугаараар OTP баталгаажуулалт.
Регистрийн дугаар + зургаар иргэний үнэмлэх баталгаажуулах (ID verification).
Малын А данс (А дансны дугаар + Мал аж ахуйн газрын тоо бүртгэлийн холбоос).
Байршлыг GPS-ээр тэмдэглэх + хүн бараа очиж болох хаяг бичих.
Банкны данс холбох (khanbank, golomt, TDB API-ууд).
Багийн захирагчийн бичгээр эсвэл QR-аар баталгаажсан гэрчилгээ upload.
Баталгаажуулалт админ талаас 24-48 цагт шалгагдана.
6.2. Бүтээгдэхүүний listing
Ангилал: Мах (үхрийн, хонины, ямааны, адууны, тэмээний), Ноос, Арьс, Сүү, Цагаан идээ.
Зураг: заавал 1+ зураг, маx 8 зураг (барааны, малын, фермийн).
Тайлбар: нас, жин, үүлдэр, бэлтгэсэн арга, тусгай шинж чанар.
Үнэ: кг-аар, ширхгээр, литрээр гэх мэт нэгжээр.
Хэмжээ, бэлэн байдал (нөөц), хамгийн багадаа захиалах хэмжээ.
Халаагч/хөргөгчтэй тээвэрлэх шаардлагатай эсэх (cold chain тэмдэглэгээ).
Хугацаа: ямар хугацаанд бэлэн болж, илгээгдэх.
Мал эмнэлгийн гэрчилгээ upload (мах, сүүний хувьд заавал).
6.3. Хайлт ба ангилал (Discovery)
Аймгаар шүүх (одоогийн UI-тай тохирох).
Сумаар нарийвчлан шүүх (advanced filter).
Ангилалаар шүүх (мах, ноос, арьс, сүү).
Үнийн range шүүлт.
Хүргэлтийн хугацаагаар эрэмбэлэх (хамгийн хурдан).
Үнэлгээгээр эрэмбэлэх (хамгийн өндөр rating-тай малчид).
Органик, гэрчилгээтэй гэх мэт tag-аар шүүх.
Газрын зургаар (map view) байршлыг харах сонголт.
6.4. Захиалга ба төлбөр
Сагсанд нэмэх, олон малчнаас нэг захиалга дотор бараа нийлүүлэх.
Төлбөрийн сонгогт: QPay, SocialPay, Khan Bank, Golomt, TDB, Карт.
Escrow систем: мөнгө малчинд шууд очихгүй, хүргэлт баталгаажтал платформ дээр хадгалагдана.
Захиалга баталгаажсаны дараа малчинд notification илгээх.
Захиалга цуцлах дүрэм: малчин 24 цагт хариу өгөхгүй бол автомат цуцална, мөнгө буцаана.
6.5. Хүргэлт ба логистик
Сонголтууд: Монгол Шуудан, хувийн карго компаниуд (Номин, Тосон, Очир Ундраа), нийтийн тээврээр.
Хамтарсан тээвэрлэлт (consolidation): тухайн аймгийн төвд цугласан барааг долоо хоногт 2 удаа УБ руу илгээх.
Tracking number автоматаар үүсгэх, real-time track хийх.
Хүргэлтийн статусын шинэчлэл: хүлээн авсан → боож бэлдсэн → илгээсэн → аймгийн төвд → УБ-д → хэрэглэгчид.
Cold chain шаардлагатай бүтээгдэхүүнд тусгай logistics хэрэгслэн холбох.
Амжилтгүй хүргэлтийн хариуцлагын бодлого (малчин vs платформ vs хэрэглэгч).
6.6. Review ба rating систем
Бүтээгдэхүүний үнэлгээ (1-5 од) + бичмэл сэтгэгдэл + зураг upload.
Малчны үнэлгээ тусдаа (харилцаа, хугацаа баримтлалт, чанар).
Буцаалтын хүсэлт илгээх, маргаан шийдвэрлэх flow.
Худалдан авагч зөвхөн хүргэгдсэн захиалга дээр үнэлгээ өгөх боломжтой (fake review-ийг бууруулах).
Malчны нийт rating, хүргэлтийн амжилт %, захиалгын тоо публик харагдана.
6.7. Малчны хяналтын самбар (Dashboard)
Ирсэн захиалгын жагсаалт, статусаар шүүлт.
Орлогын тайлан: өдөр, 7 хоног, сараар.
Commission тооцоо, платформын хураамж тодорхой харагдана.
Банкны данс руу татах хүсэлт (payout request), T+3 хоногт шилжүүлэх.
Listing-ийн статистик: хэчнээн хүн үзсэн, сагсанд нэмсэн, худалдан авсан.
6.8. Орон нутгийн координатор
Тус бүр сумд 1-2 координатор томилох.
Координаторын үүрэг: шинэ малчдад бүртгэл хийхэд туслах, бараа зургийг авах, listing үүсгэх.
Тэдэнд тусдаа admin dashboard, өөрийн referral код.
Commission: тухайн координаторын нэмсэн малчны борлуулалтын 2-3%.
7. Non-functional requirements
Performance: Page load < 2.5s (Core Web Vitals LCP), API response < 500ms.
Offline capability: малчин 2G/3G-тэй газар listing үүсгэх, sync хийх боломжтой.
Localization: Монгол, Англи хоёр хэлтэй (дараа орос, хятад нэмж болно).
Accessibility: WCAG 2.1 AA түвшинд тохирч, ахимаг настай малчдад хялбар.
Security: OWASP Top 10, PCI DSS (төлбөр), Монгол улсын хувь хүний мэдээллийн хуульд нийцсэн.
Scalability: Анх 10,000 concurrent user, дараа 100,000 руу scale.
Uptime: 99.5% SLA.
8. Үе шаттай хөгжүүлэлт (Phased Rollout)
Phase 1: MVP (6-8 долоо хоног)
Үндсэн бүртгэл + KYC.
Listing үүсгэх + browse.
Захиалга + QPay/SocialPay төлбөр.
Монгол Шуудангаар хүргэлт.
Үндсэн review систем.
3 аймгаас pilot (Архангай, Төв, Сэлэнгэ).
Phase 2: Scale-up (3-4 сар)
Бүх 21 аймгийг хамрах.
Escrow систем, маргаан шийдвэрлэх flow.
Координаторын систем.
Олон карго компаниудын integration.
Advanced search + map view.
Ахисан analytics dashboard.
Phase 3: Premium features (6-12 сар)
Subscription model — сар бүр хүргэдэг бараа (сүү, тараг).
Cold chain integration.
Bulk order (рестораны, зочид буудлын).
B2B portal.
Олон улсын экспорт (эхлээд Хятад, Солонгос — ноос, арьс).
9. Эрсдэл ба бууруулах арга
Эрүүл ахуйн эрсдэл: МХЕГ-тай зөвшилцөж, гэрчилгээ заавал upload хийлгэх.
Логистикийн эрсдэл: Карго компаниудтай SLA гэрээ байгуулах, insurance.
Малчдын технологийн саад: Координатор систем, видео заавар, call center.
Төлбөрийн маргаан: Escrow, тодорхой бодлого, арбитраж.
Хуурамч listing: KYC хатуу, admin moderation, хэрэглэгчийн гомдлын систем.
Улирлын нөлөө: Өвлийн хөргөлт, зудын үеийн стратеги.
10. Бизнесийн загвар
Орлогын эх үүсвэрүүд:
Commission: борлуулалтын 8-12% (Amazon, Shopee-той төстэй).
Promoted listing: малчид нэмэлт төлбөртэйгээр дээгүүр харагдах.
Premium features: advanced analytics, professional photography service.
Logistics margin: карго компаниудаас volume rebate.
Subscription box: сар бүрийн хүргэлтийн захиалгын 15% margin.
